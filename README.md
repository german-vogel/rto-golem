# GOLEM Tokamak Data Analysis Toolkit

A comprehensive Python toolkit for analyzing data from the GOLEM tokamak.

## Features
- **Main GUI Application**: Interactive visualization of plasma parameters
- **Spectrometry Analysis**: Automated ion identification from spectrometer data
- **MHD Analysis**: Mirnov coil data processing
- **Runaway Electrons**: Specialized analysis for runaway electron studies

## Quick Start
1. Install dependencies: `pip install -r requirements.txt`
2. Run main app: `python main_app.py`
3. Load shot data by entering shot numbers

## Project Structure
- `main_app.py` - Main Tkinter GUI application
- `spectrometry_analyzer.py` - Core spectrometry functions
- `main/MHD/` - MHD analysis modules
- `main/runaway_electrons/` - Runaway electron studies
- `main/spectroscopy/` - Spectroscopy improvements
